# FTP-Клиент для ОдноСкрипта

## Установка

`opm install ftp`


## Использование

```bsl
#Использовать ftp

Соединение = Новый FtpСоединение("example.com");

Соединение.Записать("/local/file.txt", "/dir/");

Файлы = Соединение.НайтиФайлы("/dir/", "file.txt");

Файл = Файлы[0];

РазмерФайла = Файл.Размер();

```
